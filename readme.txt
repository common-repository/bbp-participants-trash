Allows participants to trash their replies & topics.

== Description ==

A very simple plugin which allow forum participants to trash their replies and topics.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'bbPress'
3. Activate bbpress particdipant trash from your Plugins page.

= From WordPress.org =

1. Download bbPress.
2. Upload the 'bbpress-participant-trash' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate bbpress participant trash from your Plugins page.
